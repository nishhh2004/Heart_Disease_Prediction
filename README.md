# ❤️ Heart Disease Prediction System

A **Machine Learning powered Streamlit web application** that predicts the risk of heart disease based on patient medical data.

The application allows users to enter clinical information such as age, cholesterol level, blood pressure, ECG results, and other health indicators. A trained machine learning model analyzes the input data and predicts whether the patient is at **high or low risk of heart disease**.

---

## 🚀 Features

- Interactive **Streamlit Web Interface**
- Real-time **Heart Disease Prediction**
- **Dark / Light Theme Toggle**
- User-friendly medical input form
- Trained **Machine Learning Model**
- Instant prediction results

---

## 🧠 Machine Learning Workflow

The model was built using the following steps:

1. Data collection from the heart disease dataset
2. Data cleaning and preprocessing
3. Feature encoding
4. Feature scaling using **StandardScaler**
5. Model training using **Scikit-Learn**
6. Model saving using **pickle**
7. Deployment using **Streamlit**
